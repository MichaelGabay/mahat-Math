# תת-נושא 3.3: קצבים ומהירויות – חישוב "שיפוע" מילולי והגיוני

**כלל יסוד:**
$$\text{קצב שינוי} = \frac{\Delta Y}{\Delta X} = \frac{\text{הפרש ב-}Y}{\text{הפרש ב-}X}$$

**המרת יחידות מהירות:**
$$1 \text{ ק"מ/שעה} = \frac{1000 \text{ מ'}}{3600 \text{ שנ'}} = \frac{1}{3.6} \text{ מ'/שנ'}$$
