## רמה 2: תרגול שוטף ומשולב (8 תרגילים)

9. פתרו את המשוואה:
$$4(x - 3) + 2x = 18$$

10. פתרו את המשוואה:
$$3(2x + 1) - (x - 4) = 4x + 5$$

11. פתרו את המשוואה:
$$6x - 3(x - 4) = 2(x + 3) - 5$$

12. פתרו את המשוואה:
$$2(3x - 5) = 4(x + 1) - 6$$

13. פתרו את המשוואה:
$$-3(x - 2) + 4x = 2(x + 5) - 6$$

14. פתרו את המשוואה:
$$7(x + 1) - 3(2x - 1) = 2x + 14$$

15. פתרו את המשוואה:
$$4 - 2(x + 3) = 3(x - 1) + 1$$

16. פתרו את המשוואה:
$$5(2x - 3) - 4(x + 2) = 2(2x - 1) + 1$$

## רמה 3: רמת בחינת מה"ט (4 תרגילים)

17. פתרו את המשוואה (בסגנון בחינת מה"ט):
$$3[2(x - 4) + 5] - 2(3x - 1) = 4(x + 2) - [5(x - 3) + 1]$$

18. פתרו את המשוואה:
$$3(2x - 5) - 2(x - 3) = 4(x - 1) - 5(x - 3)$$

19. פתרו את המשוואה:
$$2(3x - 4) - 3(2x + 1) = 5(x - 3) - (4x - 2)$$

20. בכל הסעיפים פתרו את המשוואה:

א.
$$4(3x - 2) - 5(2x + 1) = 3(x - 4) + 2$$

ב.
$$6(x + 3) - 2(3x - 1) = 4(x - 5) + 28$$
