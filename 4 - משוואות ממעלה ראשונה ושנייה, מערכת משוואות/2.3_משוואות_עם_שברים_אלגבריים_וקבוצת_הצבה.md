## רמה 2: תרגול שוטף ומשולב (8 תרגילים)

9. קבעו תחום הגדרה ופתרו (כפלו ב-מ.מ.ק):
$$\frac{5}{x - 1} - \frac{3}{x + 1} = \frac{2}{x^2 - 1}$$

10. קבעו תחום הגדרה ופתרו:
$$\frac{1}{x - 3} + \frac{2}{x + 3} = \frac{9}{x^2 - 9}$$

11. קבעו תחום הגדרה ופתרו:
$$\frac{x}{x - 2} - \frac{4}{x + 2} = \frac{8}{x^2 - 4}$$

12. קבעו תחום הגדרה ופתרו:
$$\frac{3}{x - 5} - \frac{1}{x + 5} = \frac{20}{x^2 - 25}$$

13. קבעו תחום הגדרה ופתרו (שימו לב לאפשרות של זהות):
$$\frac{1}{x - 2} + \frac{2}{x + 2} = \frac{3x - 2}{x^2 - 4}$$

14. קבעו תחום הגדרה ופתרו:
$$\frac{2x}{x - 4} = \frac{8}{x - 4} + 3$$

15. קבעו תחום הגדרה ופתרו:
$$\frac{x + 3}{x - 1} - 2 = \frac{2x}{x - 1} - 5$$

16. קבעו תחום הגדרה ופתרו:
$$\frac{3x + 1}{x + 2} = \frac{2x - 4}{x + 2} + 2$$

## רמה 3: רמת בחינת מה"ט (4 תרגילים)

17. בסגנון בחינת מה"ט – שנת 2023:

א. פשטו את הביטוי הבא בצורת פירוק לגורמים:
$$\frac{2x}{x - 4} \div \frac{1}{2x - 8}$$

ב. מהו ערך הביטוי כאשר $x = 3$?

18. קבעו תחום הגדרה ופתרו:
$$\frac{3}{x - 2} + \frac{5}{x + 2} = \frac{2x + 14}{x^2 - 4}$$

19. קבעו תחום הגדרה ופתרו:
$$\frac{5x}{x^2 - 4} - \frac{1}{x - 2} = \frac{3}{x + 2}$$

20. בכל הסעיפים קבעו תחום הגדרה ופתרו:

א.
$$\frac{x + 1}{x - 3} + \frac{2}{x + 3} = \frac{x^2 + 3}{x^2 - 9}$$

ב.
$$\frac{2}{x - 1} - \frac{3}{x + 2} = \frac{x - 7}{x^2 + x - 2}$$
