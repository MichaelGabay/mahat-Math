# תת-נושא 4.1: הכרת המבנה של משוואה ריבועית וזיהוי המקדמים

---

## רמה 1: בניית ביטחון (8 תרגילים)

1. זהו את המקדמים $a$, $b$, $c$ במשוואה הריבועית:
$$3x^2 + 5x - 2 = 0$$

2. זהו את המקדמים:
$$x^2 - 7x + 10 = 0$$

3. זהו את המקדמים:
$$2x^2 - 8 = 0$$

4. זהו את המקדמים:
$$4x^2 + 9x = 0$$

5. זהו את המקדמים:
$$-x^2 + 6x - 5 = 0$$

6. סדרו את המשוואה בצורה הסטנדרטית ($ax^2 + bx + c = 0$) וזהו את המקדמים:
$$5x - 3 = -x^2$$

7. סדרו וזהו את המקדמים:
$$2x^2 = 10x - 8$$

8. סדרו וזהו את המקדמים:
$$3x = x^2 + 4$$

---

## רמה 2: תרגול שוטף ומשולב (8 תרגילים)

9. בדקו אם הערך הנתון הוא פתרון של המשוואה $x^2 - 5x + 6 = 0$:

א. $x = 2$

ב. $x = 3$

ג. $x = 1$

10. בדקו אם הערך הנתון הוא פתרון של המשוואה $2x^2 - 3x - 5 = 0$:

א. $x = -1$

ב. $x = \frac{5}{2}$

11. סדרו וזהו את המקדמים:
$$(x + 3)(x - 5) = 0$$

12. סדרו וזהו את המקדמים:
$$(2x - 1)(x + 4) = 2x - 1$$

13. סדרו וזהו את המקדמים:
$$x(x - 7) = 3x - 15$$

14. חשבו את הדיסקרימיננטה $\Delta = b^2 - 4ac$ של המשוואה:
$$2x^2 - 5x + 3 = 0$$

15. חשבו את הדיסקרימיננטה:
$$x^2 + 4x + 4 = 0$$

16. חשבו את הדיסקרימיננטה:
$$3x^2 - 2x + 5 = 0$$

---

## רמה 3: רמת בחינת מה"ט (4 תרגילים)

17. סדרו את המשוואה לצורה הסטנדרטית, זהו את המקדמים, ואחר כך חשבו את הדיסקרימיננטה:
$$8 - \frac{(x + 4)(x - 8)}{3} = -2x$$

18. פשטו וסדרו לצורה הסטנדרטית:
$$4 + (x + 1) = 12(x - 1) - 2(x + 1)(x - 1)$$

19. פשטו וסדרו לצורה הסטנדרטית:
$$9 + x - 2(2 + x) = 3(4 - x^2) - (2 - x)$$

20. בכל הסעיפים סדרו לצורה הסטנדרטית וזהו את $a$, $b$, $c$:

א.
$$(x - 3)^2 = 2x + 1$$

ב.
$$\frac{(x + 2)(x - 2)}{3} = x - 4$$
