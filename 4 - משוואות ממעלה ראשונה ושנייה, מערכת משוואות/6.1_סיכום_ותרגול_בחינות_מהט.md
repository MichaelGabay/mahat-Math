## רמה 1: בניית ביטחון (8 תרגילים)

8 שאלות גישור – זיהוי סוג המשוואה ובחירת שיטת הפתרון הנכונה.

1. זהו את סוג המשוואה (ממעלה ראשונה / ריבועית / מערכת) ופתרו:
$$5x - 3 = 2x + 9$$

2. זהו את סוג המשוואה ופתרו:
$$x^2 - 7x + 12 = 0$$

3. זהו את סוג המשוואה ופתרו:
$$x^2 = 4x$$

4. זהו את סוג המשוואה ופתרו:
$$2(3x - 1) = 4(x + 3) - 2$$

5. זהו את סוג המשוואה ופתרו:
$$x^2 + 5x + 6 = 0$$

6. זהו את סוג המשוואה ופתרו:
$$\frac{x + 1}{3} = \frac{x - 2}{2}$$

7. זהו את סוג המערכת ופתרו:
$$y = 2x - 1$$
$$3x + y = 9$$

8. זהו את סוג המערכת ופתרו:
$$2x + 3y = 13$$
$$x - y = -1$$

## רמה 2: תרגול שוטף ומשולב (8 תרגילים)

9. פתרו משוואה עם שברים:
$$\frac{3x - 1}{4} + \frac{x + 2}{8} = \frac{2x + 3}{2}$$

10. פתרו משוואה ריבועית לאחר סידור:
$$(x + 2)(x - 3) = x - 6$$

11. פתרו מערכת בשיטת השוואת מקדמים:
$$5x - 2y = 8$$
$$3x + 4y = 10$$

12. פתרו:
$$\frac{3}{x - 1} - \frac{2}{x + 1} = \frac{1}{x^2 - 1}$$

13. פתרו:
$$2x^2 - 7x + 3 = 0$$

14. פתרו מערכת:
$$y = x^2 - 4x + 4$$
$$y = x - 2$$

15. פתרו מערכת לינארית בשיטת ההצבה:
$$4x - y = 7$$
$$2x + 3y = 5$$

16. פתרו:
$$\frac{2x + 1}{3} - \frac{x - 1}{6} = \frac{x + 4}{2}$$
