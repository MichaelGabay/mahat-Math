## רמה 1: בניית ביטחון (8 תרגילים)

1. בדלו את $y$ מהמשוואה הלינארית ופתרו:
$$y = x^2 - x - 2$$
$$y = 3x - 2$$

2. פתרו:
$$y = x^2 + 2x$$
$$y = 3x$$

3. פתרו:
$$y = x^2 - 4x + 4$$
$$y = x - 2$$

4. פתרו:
$$y = x^2 + x - 6$$
$$y = 2x$$

5. פתרו:
$$y = x^2 - 6x + 5$$
$$y = x - 1$$

6. פתרו:
$$y = x^2 - 1$$
$$y = x + 1$$

7. פתרו:
$$y = x^2 + 3x$$
$$y = 4x + 6$$

8. פתרו:
$$y = x^2 - 9$$

## רמה 2: תרגול שוטף ומשולב (8 תרגילים)

9. בדלו את $y$ מהמשוואה הלינארית ואחר כך הציבו:
$$y = x^2 + 3x - 4$$
$$y - x = 4$$

10. פתרו:
$$y = x^2 - 1$$
$$y - x = 1$$

11. פתרו:
$$y = x^2 - 8$$
$$y = 2x$$

12. פתרו:
$$y = x^2 - 4x + 2$$
$$y + x = 6$$

13. פתרו:
$$y = x^2 - 3x + 6$$
$$y - 2x = 0$$

14. פתרו:
$$y = x^2 - 5x + 7$$
$$y - x = 2$$

15. פתרו:
$$y = x^2 - 6x + 10$$
$$y + x = 4$$

16. פתרו:
$$y = x^2 - 2x - 1$$
$$y - x = -3$$

## רמה 3: רמת בחינת מה"ט (4 תרגילים)

17. פתרו ובדקו שני הפתרונות (בסגנון בחינת מה"ט – שנת 2024):
$$y = x^2 + 2x - 3$$
$$y - 3x = -1$$

18. פתרו ובדקו (בסגנון בחינת מה"ט – שנת 2025):

**הערה**: פשטו תחילה את המשוואה הראשונה וקבעו את סוג כל משוואה לפני הפתרון.
$$3(2x - y) - 10 = 5x - 3$$
$$y = -2x + 7$$

19. פתרו ובדקו:
$$y = x^2 - 4x + 4$$
$$y = x - 2$$

20. בכל הסעיפים פתרו ובדקו:

א.
$$y = x^2 - 3x$$
$$y = 4x - 12$$

ב.
$$y = x^2 + x - 6$$
$$y = 2x - 4$$
