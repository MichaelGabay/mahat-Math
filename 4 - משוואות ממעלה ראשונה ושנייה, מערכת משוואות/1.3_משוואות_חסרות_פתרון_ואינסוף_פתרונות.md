## רמה 1: בניית ביטחון (8 תרגילים)

קבעו בכל תרגיל: האם למשוואה פתרון יחיד, אין פתרון, או אינסוף פתרונות.

1. פתרו או קבעו:
$$2x + 5 = 2x + 8$$

2. פתרו או קבעו:
$$3x - 4 = 3x - 4$$

3. פתרו או קבעו:
$$5x + 1 = 5x - 1$$

4. פתרו או קבעו:
$$4x - 3 = 4x - 3$$

5. פתרו או קבעו:
$$-x + 7 = -x + 7$$

6. פתרו או קבעו:
$$6x + 2 = 6x + 5$$

7. פתרו או קבעו:
$$-2x - 9 = -2x - 9$$

8. פתרו או קבעו:
$$x + 10 = x - 1$$

## רמה 2: תרגול שוטף ומשולב (8 תרגילים)

9. פתחו סוגריים ואחר כך קבעו את סוג המשוואה:
$$2(x + 3) = 2x + 6$$

10. פתחו סוגריים וקבעו:
$$3(x - 4) = 3x - 11$$

11. פתחו סוגריים וקבעו:
$$4(x + 2) - 3 = 4x + 5$$

12. פתחו סוגריים וקבעו:
$$5(2x - 1) - 3x = 7x - 5$$

13. פתחו סוגריים וקבעו:
$$2(3x + 1) - 4x = 2(x + 1)$$

14. פתחו סוגריים וקבעו:
$$3(2x - 5) + 15 = 6x$$

15. פתחו סוגריים וקבעו:
$$-(2x - 8) = -2x + 9$$

16. פתחו סוגריים וקבעו:
$$4(x - 1) + 3 = 2(2x + 1) - 3$$

## רמה 3: רמת בחינת מה"ט (4 תרגילים)

17. פתחו את הסוגריים ואחר כך קבעו אם יש פתרון יחיד, אין פתרון, או אינסוף פתרונות. אם יש פתרון יחיד – מצאו אותו:
$$3(2x - 4) - (x + 5) = 5(x - 1) - 12$$

18. פתחו את הסוגריים ואחר כך קבעו:
$$2(3x + 1) - 4(x - 3) = 3x + 8$$

19. פתחו את הסוגריים ואחר כך קבעו:
$$4(x - 3) + 2(x + 1) = 3(2x - 5) + 7$$

20. בכל סעיף, קבעו את סוג המשוואה ופתרו אם אפשר:

א.
$$5(2x - 3) - (x + 1) = 9(x - 1) + 1$$

ב.
$$3(4x - 2) - (6x - 1) = 2(3x - 1) - 3$$
