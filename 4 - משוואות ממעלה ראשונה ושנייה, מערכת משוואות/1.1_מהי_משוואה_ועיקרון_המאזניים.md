# תת-נושא 1.1: מהי משוואה? עיקרון המאזניים וביצוע פעולות זהות על שני האגפים

---

## רמה 1: בניית ביטחון (8 תרגילים)

1. פתרו את המשוואה על ידי הוספה או חיסור זהים משני האגפים:
$$x + 7 = 15$$

2. פתרו את המשוואה:
$$x - 4 = 9$$

3. פתרו את המשוואה על ידי כפל או חילוק זהים משני האגפים:
$$3x = 21$$

4. פתרו את המשוואה:
$$\frac{x}{5} = 4$$

5. פתרו את המשוואה:
$$x + 12 = 3$$

6. פתרו את המשוואה:
$$-6x = 18$$

7. פתרו את המשוואה:
$$x - 8 = -3$$

8. פתרו את המשוואה:
$$\frac{x}{-4} = 7$$

## רמה 2: תרגול שוטף ומשולב (8 תרגילים)

9. פתרו את המשוואה (שתי פעולות על האגפים):
$$4x - 3 = 17$$

10. פתרו את המשוואה:
$$-5x + 2 = -13$$

11. פתרו את המשוואה (העבירו אגפים):
$$3x + 8 = x - 4$$

12. פתרו את המשוואה:
$$7x - 1 = 4x + 11$$

13. פתרו את המשוואה:
$$-2x + 9 = 5x - 5$$

14. פתרו את המשוואה:
$$6 - 3x = 4x - 8$$

15. פתרו את המשוואה (פתחו סוגריים תחילה):
$$2(x + 5) = 18$$

16. פתרו את המשוואה:
$$-(x - 3) = 2x + 9$$

---

## רמה 3: רמת בחינת מה"ט (4 תרגילים)

17. פתרו את המשוואה:
$$3(2x - 4) - (x + 5) = 2(x - 3) + 4$$

18. פתרו את המשוואה:
$$4(x - 3) + 2(x + 1) = 3(x + 2) - 4$$

19. פתרו את המשוואה:
$$5(2x + 1) - 3(x - 4) = 2(4x - 1) + 5$$

20. בכל הסעיפים פתרו את המשוואה:

א.
$$8x - 3(x - 2) = 2(x + 7) + 1$$

ב.
$$3(2x - 1) - 2(x + 4) = 2(x - 5) + 3x$$
