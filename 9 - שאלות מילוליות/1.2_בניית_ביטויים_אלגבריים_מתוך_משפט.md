# יחידה 1.2 – בניית ביטויים אלגבריים פשוטים מתוך משפט

---

## מבוא

**הכלל:** בחר נעלם, הגדר אותו במפורש, ואחר כך בנה ביטוי עבור כל כמות אחרת.

> **דוגמה:**
> יהי $x$ = מחיר מוצר א' (ב-₪).
> מוצר ב' עולה 30 ₪ יותר ממוצר א'.
> ביטוי למחיר מוצר ב':
> $$x + 30$$

---

## רמה 1 – נעלם אחד, ביטוי פשוט (תרגילים 1–8)

**1.** יהי $x$ = גיל אחות.
אח בן 6 שנים יותר מהאחות.
כתוב ביטוי לגיל האח.

**2.** יהי $n$ = מספר הכרטיסים שיש לדן.
לאורי יש פי-3 כרטיסים מדן.
כתוב ביטוי למספר הכרטיסים של אורי.

**3.** יהי $p$ = מחיר עוגה (ב-₪).
מחיר לחם הוא 9 ₪ פחות ממחיר עוגה.
כתוב ביטוי למחיר הלחם.

**4.** יהי $m$ = משקל ארגז א' (בק"ג).
ארגז ב' קל פי-4 מארגז א'.
כתוב ביטוי למשקל ארגז ב'.

**5.** יהי $t$ = זמן נסיעה ברכבת (בדקות).
נסיעה באוטובוס ארוכה ב-20 דקות מנסיעה ברכבת.
כתוב ביטוי לזמן הנסיעה באוטובוס.

**6.** יהי $s$ = מספר התלמידים בכיתה א'.
בכיתה ב' יש $\dfrac{1}{3}$ מכמות התלמידים בכיתה א'.
כתוב ביטוי למספר התלמידים בכיתה ב'.

**7.** יהי $d$ = מרחק מהבית לבית הספר (בק"מ).
המרחק מהבית לסופרמרקט גדול פי-2.5 מהמרחק לבית הספר.
כתוב ביטוי למרחק מהבית לסופרמרקט.

**8.** יהי $h$ = גובה בניין (במטרים).
גובה הגדר שמסביבו הוא 3 מטרים פחות מ-$\dfrac{1}{5}$ גובה הבניין.
כתוב ביטוי לגובה הגדר.

---

## רמה 2 – שני נעלמים, אחוזים וביטויים משולבים (תרגילים 9–16)

**9.** יהי $x$ = מחיר כרטיס לסרט (ב-₪).
כרטיס לתיאטרון עולה 45 ₪ יותר מכרטיס לסרט.
כתוב ביטוי ל:

א. מחיר כרטיס לתיאטרון.

ב. סכום מחירי שני הכרטיסים.

**10.** יהי $a$ = מחיר מוצר (ב-₪).
המוצר עלה ב-12%.

א. כתוב ביטוי למחיר החדש (לאחר העלייה).

ב. לאחר שבוע, ניתנה הנחה של 5% על המחיר החדש. כתוב ביטוי למחיר הסופי.

**11.** יהי $b$ = מספר הבנים בכיתה.
מספר הבנות גדול ב-4 ממספר הבנים.

א. כתוב ביטוי למספר הבנות.

ב. כתוב ביטוי לכלל התלמידים בכיתה.

ג. אם הכיתה כוללת 30 תלמידים, בנה משוואה (אל תפתור).

**12.** יהי $x$ = אורך המלבן (בס"מ).
רוחב המלבן קטן ב-3 ס"מ מהאורך.

א. כתוב ביטוי לרוחב.

ב. כתוב ביטוי להיקף המלבן.

ג. כתוב ביטוי לשטח המלבן.

**13.** שני שותפים: יוסי ודוד.
יוסי השקיע $x$ ₪.
דוד השקיע פי-$\dfrac{3}{2}$ מיוסי.

א. כתוב ביטוי להשקעת דוד.

ב. כתוב ביטוי לסך ההשקעה המשותפת.

ג. יוסי קיבל 40% מהרווח הכולל $r$. כתוב ביטוי לרווח של יוסי.

**14.** יהי $p$ = מחיר ק"ג תפוחים (ב-₪).
מחיר ק"ג אגסים יקר ב-3 ₪ מתפוחים.
מיכל קנתה 2 ק"ג תפוחים ו-3 ק"ג אגסים.
כתוב ביטוי לסכום ששילמה.

**15.** יהי $n$ = מספר השאלות שענה דניאל נכון.
מספר השאלות שטעה הוא $\dfrac{1}{4}$ ממספר השאלות שענה נכון.
המבחן כלל 25 שאלות בסך הכל.

א. כתוב ביטוי למספר השאלות שטעה.

ב. בנה משוואה (אל תפתור).

**16.** בחנות יש שני סוגי מוצרים: מוצר A במחיר $x$ ₪ ומוצר B במחיר $y$ ₪.

| מוצר | מחיר ליחידה | כמות | עלות |
|------|------------|------|------|
| A | $x$ | 4 | $4x$ |
| B | $y$ | 7 | $7y$ |
| **סה"כ** | | | |

א. השלם את העמודה "עלות" עבור מוצר B ועבור שורת הסה"כ.

ב. ידוע כי $y = x + 8$. כתוב את הסכום הכולל בעזרת $x$ בלבד.

## רמה 3 – שאלות מהט מלאות (תרגילים 17–20)

**17.** ספר לימוד עולה $x$ ₪.
מחברת עולה 12 ₪ פחות מהספר.
קלמר עולה פי-2 מהמחברת.

| פריט | מחיר |
|------|------|
| ספר | $x$ |
| מחברת | |
| קלמר | |

א. מלא את הטבלה עם ביטויים ב-$x$.

ב. נועה קנתה 3 ספרים, 5 מחברות ו-2 קלמרים. כתוב ביטוי לסכום ששילמה.

ג. הפשט את הביטוי.

**18.** בטיול של בית ספר, מחיר כרטיס אוטובוס לתלמיד הוא $t$ ₪.
מחיר כרטיס למורה יקר ב-30% מכרטיס תלמיד.
בטיול השתתפו 40 תלמידים ו-3 מורים.

| | מחיר לכרטיס | כמות | עלות |
|---|---|---|---|
| תלמיד | $t$ | 40 | |
| מורה | | 3 | |
| **סה"כ** | | | |

א. מלא את הטבלה.

ב. פשט את הביטוי לעלות הכוללת.

ג. אם הסכום הכולל הוא 4,390 ₪, בנה משוואה (אל תפתור).

**19.** לגן יש שתי ערוגות פרחים.
ערוגה א' היא מלבן שאורכו $x$ מטרים ורוחבו 4 מטרים.
ערוגה ב' היא מלבן שאורכו גדול ב-3 מטרים מאורך ערוגה א', ורוחבו קטן ב-1 מטר מרוחב ערוגה א'.

א. כתוב ביטויים לאורך ולרוחב של ערוגה ב'.

ב. כתוב ביטוי לשטח ערוגה ב'.

ג. כתוב ביטוי לשטח הכולל של שתי הערוגות.

ד. אם שטח הכולל הוא 58 מ"ר, בנה משוואה (אל תפתור).

**20.** חנות מוכרת שני גדלי קופסאות שוקולד.
קופסה קטנה מכילה $n$ יחידות שוקולד ועולה 18 ₪.
קופסה גדולה מכילה פי-$\dfrac{5}{3}$ יחידות מהקופסה הקטנה ועולה 28 ₪.

| | יחידות | מחיר | כמות שנקנתה | עלות |
|---|---|---|---|---|
| קטנה | $n$ | 18 | 6 | |
| גדולה | | 28 | 4 | |
| **סה"כ** | | | | |

א. מלא את הטבלה.

ב. כתוב ביטוי לסך יחידות השוקולד שנקנו.

ג. כמה יותר יחידות שוקולד יש בקופסאות הגדולות מאשר בקופסאות הקטנות? כתוב ביטוי.
