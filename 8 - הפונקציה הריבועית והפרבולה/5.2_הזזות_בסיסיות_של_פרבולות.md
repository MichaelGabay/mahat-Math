# יחידה 5.2 – הזזות בסיסיות של פרבולות (ימינה/שמאלה, למעלה/למטה)

## רמה 1: בניית ביטחון (8 תרגילים)

1. כתוב את משוואת הפרבולה
$y = x^2$
לאחר הזזה:
א. $3$ יחידות למעלה
ב. $5$ יחידות למטה

2. כתוב את משוואת הפרבולה
$y = x^2$
לאחר הזזה:
א. $2$ יחידות ימינה
ב. $4$ יחידות שמאלה

3. כתוב את משוואת הפרבולה
$y = x^2$
לאחר הזזה של $3$ יחידות ימינה ו-$4$ יחידות למעלה.

4. כתוב את משוואת הפרבולה
$y = x^2$
לאחר הזזה של $1$ יחידה שמאלה ו-$2$ יחידות למטה.

5. נתונה הפרבולה
$y = (x-3)^2 + 2$.
א. מהו קודקוד הפרבולה?
ב. כיצד הזזת את
$y = x^2$
כדי לקבל פרבולה זו?

6. נתונה הפרבולה
$y = (x+2)^2 - 5$.
מהו קודקוד הפרבולה?

7. נתונה הפרבולה
$y = (x-1)^2$.
א. מהו קודקוד הפרבולה?
ב. מהי נקודת החיתוך עם ציר $y$?

8. כתוב בצורת הזזה
$y = (x-h)^2 + k$:
$y = x^2 - 6x + 11$
(רמז: השלם לריבוע)

## רמה 2: תרגול שוטף ומשולב (8 תרגילים)

9. נתונה הפרבולה
$y = x^2$.
הזזנו אותה $3$ יחידות ימינה ו-$4$ יחידות למטה.
א. כתוב את משוואת הפרבולה החדשה בצורת הזזה.
ב. פתח לצורה הסטנדרטית.
ג. מהו הקודקוד?

10. נתונה הפרבולה
$y = (x-5)^2$.
א. מהו קודקוד הפרבולה?
ב. מהי נקודת החיתוך עם ציר $y$?
ג. האם הפרבולה חותכת את ציר $x$? (כמה פעמים?)

11. השלם לריבוע ומצא את הקודקוד:
$y = x^2 - 4x + 7$

12. השלם לריבוע ומצא את הקודקוד:
$y = x^2 + 6x + 5$

13. נתונה הפרבולה
$y = x^2$
הוזזה $a$ יחידות ימינה ו-$b$ יחידות למעלה.
ידוע שהקודקוד נמצא ב-$(3, -2)$.
כתוב את משוואת הפרבולה.

14. נתונה הפרבולה
$y = (x-5)^2$
ופרבולה שנייה
$y = -x^2 + 8x - 11$.
א. מצא את נקודות החיתוך של שתי הפרבולות.
ב. מהי הנקודה הגבוהה יותר מבין שתי נקודות החיתוך?

15. כתוב את משוואות הפרבולות הבאות בצורת הזזה
$y = a(x-h)^2 + k$
ומצא את הקודקוד:
א.
$y = x^2 - 8x + 12$
ב.
$y = -x^2 + 6x - 5$

16. נתונות שתי פרבולות:
$y = x^2 - 6x + 13$
ו-$y = -x^2 + 8x - 7$.
א. כתוב כל אחת בצורת הזזה.
ב. מצא את הקודקוד של כל אחת.
ג. האם שתי הפרבולות חותכות זו את זו? (השוה את המשוואות)

## רמה 3: רמת בחינת מה"ט (4 תרגילים)

17. מקור: מה"ט 2023

נתונה הפרבולה
$$y = x^2$$
הוזזה $3$ יחידות ימינה ו-$4$ יחידות למעלה.

א. כתוב את משוואת הפרבולה החדשה.

ב. מהו קודקוד הפרבולה החדשה?

ג. מצא את נקודת החיתוך של הפרבולה החדשה עם ציר $y$.

ד. מצא את נקודות החיתוך של הפרבולה החדשה עם ציר $x$ (אם יש).

18. מקור: מה"ט 2024

נתונות שתי פרבולות:
$$y = -x^2 + 8x - 11$$
$$y = (x-5)^2$$

א. כתוב את הפרבולה הראשונה בצורת הזזה.

ב. כתוב את קודקוד כל פרבולה.

ג. מצא את נקודות החיתוך $A$ ו-$B$ של שתי הפרבולות.

ד. מצא את מרחק $AB$.

19. מקור: מה"ט 2025

נתונות שתי פרבולות:
$$y = x^2 - 6x + 13$$
$$y = -x^2 + 8x - 7$$

א. כתוב כל פרבולה בצורת הזזה ומצא את קודקודיהן.

ב. מצא את נקודות החיתוך של שתי הפרבולות.

ג. האם שתי הפרבולות חותכות את ציר $x$? (השתמש בדיסקרימיננטה)

ד. ציין את ציר הסימטריה של כל פרבולה.

20. מקור: מה"ט 2024

נתונה הפרבולה
$$y = x^2$$
שהוזזה ימינה ב-$5$ יחידות ונגיעה בנקודה $(a, b)$ עם הישר
$$y = 3x + 4$$

א. כתוב את משוואת הפרבולה המוזזת.

ב. מצא את נקודות החיתוך של הפרבולה המוזזת עם הישר.

ג. מצא את קודקוד הפרבולה המוזזת.
