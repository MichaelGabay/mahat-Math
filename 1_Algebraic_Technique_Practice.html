$$\frac{32x}{\frac{x^4}{2}} = 32x \cdot \frac{2}{x^4} = \frac{64x}{x^4} = \frac{64}{x^3} = 64x^{-3}$$

$$x^3 + 2x^2 - x - 2 = x^2(x+2) - 1(x+2) = (x^2-1)(x+2) = (x-1)(x+1)(x+2)$$

פירוק מכנה:

$$x^2 - 1 = (x-1)(x+1)$$

צמצום:

$$\frac{(x-1)(x+1)(x+2)}{(x-1)(x+1)} = x + 2 \quad (x \neq \pm 1)$$

הצבה:

$$x = -2 \;\Rightarrow\; -2 + 2 = 0 \;\checkmark$$

פתרון תרגיל 19:
מציבים:

$$l = 2w - 1$$

בנוסחת ההיקף:

$$2(2w - 1) + 2w = 36$$
$$4w - 2 + 2w = 36 \;\Rightarrow\; 6w = 38 \;\Rightarrow\; w = \frac{38}{6} = 6\frac{1}{3}$$

פתרון תרגיל 20:
הביטוי שווה:

$$-x^2 + 4$$

מכיוון ש:

$$x^2 \geq 0 \;\Rightarrow\; -x^2 \leq 0 \;\Rightarrow\; -x^2 + 4 \leq 4$$

לכן הביטוי לעולם לא יעלה על 4, ולכן לא ניתן שיהיה גדול מ-4.
