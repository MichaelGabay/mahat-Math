# תת-נושא 2.2: "תפקיד האות" – מהו משתנה, מהו ביטוי אלגברי, והצבת מספרים בביטויים

---

## רמה 1: בניית ביטחון (8 תרגילים)

**1.** הצב $x = 3$ בביטוי $2x + 5$.

**2.** הצב $x = -2$ בביטוי $3x - 1$.

**3.** הצב $a = 4$ בביטוי $a^2 - 3a + 2$.

**4.** הצב $t = 0$ בביטוי $5t^2 - 3t + 7$.

**5.** הצב $x = 1$ בביטוי $x^3 - x^2 + x - 1$.

**6.** הצב $a = 2,\ b = 3$ בביטוי $2a + 3b$.

**7.** הצב $x = -1$ בביטוי $x^4 + x^3 + x^2 + x + 1$.

**8.** הצב $y = \dfrac{1}{2}$ בביטוי $4y + 2$.

---

## רמה 2: תרגול שוטף ומשולב (8 תרגילים)

**9.** הצב $x = -3$ בביטוי $x^2 - 5x + 6$.

**10.** הצב $a = 2,\ b = -1$ בביטוי $a^2 - 2ab + b^2$.

**11.** הצב $x = -2$ בביטוי $3x^3 - 2x^2 + x - 5$.

**12.** הצב $a = 3,\ b = -2$ בביטוי $\dfrac{a^2 - b^2}{a + b}$.

**13.** הצב $x = \dfrac{1}{3}$ בביטוי $9x^2 - 6x + 1$.

**14.** הצב $t = -\dfrac{1}{2}$ בביטוי $4t^2 + 4t + 1$.

**15.** הצב $x = 2,\ y = -3$ בביטוי $x^2y - xy^2$.

**16.** הצב $a = -1,\ b = 2,\ c = -3$ בביטוי $a + b \cdot c - a \cdot b$.

---

## רמה 3: רמת בחינת מה"ט (4 תרגילים)

**17.** הביטוי $P(x) = 2x^2 - 5x + 3$. חשב את $P(2)$, $P(-1)$ ו-$P\!\left(\dfrac{1}{2}\right)$.

**18.** עבור אילו ערכי $x$ שלמים הביטוי $x^2 - 4$ מקבל ערך שלילי?

**19.** הצב $a = 4,\ y = 2$ בביטוי $\dfrac{2a + 2y}{a + y} - \dfrac{5a^2 - 5ay}{a - y}$. חשב את הערך המספרי.

**20.** שטח מלבן שאורכו $3x - 2$ ורוחבו $x + 4$ הוא ביטוי אלגברי. פתח את הביטוי ולאחר מכן חשב את הערך עבור $x = 5$.
