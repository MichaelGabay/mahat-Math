## רמה 3: רמת בחינת מה"ט (4 תרגילים)

**17.** אם $A = 3x^2 - 2x + 1$ ו-$B = x^2 + 5x - 4$, חשב ופשט את $2A - 3B$.

**18.** הפולינום $P(x) = ax^2 + bx + c$ מקיים את התנאים: $P(0) = 3$, $P(1) = 7$, $P(-1) = 1$. מצא את הערכים $a$, $b$, $c$.

**19.** מהי דרגת הפולינום המתקבל מחיסור $P(x) = 5x^4 - 3x^2 + 1$ מתוך $Q(x) = 5x^4 + 2x^3 - x + 7$? חשב את $Q(x) - P(x)$.

**20.** אם $A + B = 5x^2 - 3x + 2$ ו-$A - B = x^2 + x - 4$, מצא את $A$ ואת $B$.
